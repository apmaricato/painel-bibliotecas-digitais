# 📚 Painel Nacional de Bibliotecas Digitais

Mapeamento nacional das bibliotecas digitais assinadas por instituições de ensino superior do Brasil.

**📋 [Contribuir com dados](COLE_AQUI_O_LINK_DO_FORMULARIO)**

## Sobre o projeto

Este painel reúne dados coletados colaborativamente por bibliotecários de IES de todo o Brasil, mapeando quais bibliotecas digitais cada instituição assina atualmente.

- **178 instituições** mapeadas
- **634 assinaturas** registradas  
- **82 bibliotecas digitais** distintas
- **27 estados** cobertos

## Estrutura do projeto

| Arquivo | Descrição |
|---------|-----------|
| `index.html` | Landing page pública (GitHub Pages) |
| `limpeza_dados.gs` | Script de limpeza automática (Google Apps Script) |
| `aliases_bibliotecas.csv` | Padronização de nomes de bibliotecas |
| `aliases_instituicoes.csv` | Padronização de nomes de instituições |
| `COMO_INSTALAR.md` | Guia de instalação completo |

## Tecnologias

- **Coleta**: Google Forms → Google Sheets
- **Limpeza**: Google Apps Script (automático a cada resposta)
- **Visualização**: Google Looker Studio
- **Landing page**: HTML estático no GitHub Pages
