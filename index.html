O Projeto

Por que este painel existe?

O acesso a conteúdo acadêmico digital no Brasil é fragmentado e opaco. Não existe um registro público consolidado de quais bibliotecas digitais cada IES assina, dificultando benchmarking, negociações coletivas e políticas de acesso ao conhecimento.

Este painel reúne dados coletados por meio de formulário colaborativo, respondido voluntariamente por profissionais de bibliotecas de todo o Brasil. Os dados são públicos, abertos e atualizados periodicamente.

Desenvolvido por Pedro Maricato - Bibliotecário CRB 8 - 6922, com o objetivo de mapear, comparar e qualificar o acesso às fontes digitais nas instituições de ensino superior brasileiras.

📋 Coleta de dados

Formulário Google respondido por bibliotecários das IES. Cada instituição informa quais bibliotecas digitais assina atualmente.

🔄 Atualização

Os dados são processados automaticamente na aba "Dados Limpos" da planilha. O painel web lê esses dados atualizados em tempo real no carregamento da página.

🔓 Acesso aberto

Todos os dados são públicos. Pesquisadores e gestores podem acessar, filtrar e baixar as informações diretamente pela tabela do painel.

Metodologia

Como os dados chegam ao painel

Um pipeline simples e transparente, do formulário ao painel, sem intervenção manual após cada resposta.

1

Bibliotecário preenche o formulário

Um representante da biblioteca de cada IES informa o tipo de instituição, estado e quais bibliotecas digitais são assinadas atualmente. Leva menos de 2 minutos.

2

Resposta cai automaticamente na planilha

O Google Forms salva cada resposta diretamente em uma planilha Google Sheets — sem nenhuma ação manual necessária.

3

Script de limpeza roda automaticamente

Um script (Google Apps Script) é acionado imediatamente: padroniza nomes, corrige variações de grafia, mantém somente a resposta mais recente de cada instituição e grava os dados limpos.

4

Painel reflete os dados atualizados

Ao abrir ou atualizar esta página, os dados limpos são puxados da planilha diretamente via API e renderizados nos gráficos de forma instantânea.
